# TerminalRPG

Um jogo que simula terminal feito em HTML, CSS e JavaScript, onde você digita comandos para interagir e realizar ações.

## Funcionalidades
- Simula um terminal
- Eventos baseados em comandos digitados

## Como rodar
1. Baixe os arquivos
2. Abra o `index.html` no navegador
